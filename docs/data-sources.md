# 📊 Data Sources Overview

This document provides an overview of all external and internal data sources used by the platform.  
These data sources feed the ETL pipelines, valuation engine, backend API, and frontend UI.

---

## 1. Market Price Data

### Source
External market data provider (e.g., Alpha Vantage, Finnhub, EODHD, Polygon, or Yahoo Finance).

### Data Retrieved
- Daily open, high, low, close prices  
- Trading volume  
- Ticker metadata  
- Historical price series  

### Usage
- Discount calculation  
- Screener ranking  
- Price vs fair value charts  
- Relative valuation metrics (e.g., FCF yield)  

---

## 2. Fundamentals Data

### Source
External financial data provider (e.g., Alpha Vantage, Finnhub, EODHD).

### Data Retrieved
- Revenue  
- Net income  
- EBITDA  
- EBIT  
- Free cash flow  
- Total debt  
- Total equity  
- Shares outstanding  

### Usage
- DCF valuation  
- Relative valuation  
- Growth calculations  
- Profitability metrics  

---

## 3. Company Metadata

### Source
External company information API or static dataset.

### Data Retrieved
- Company name  
- Ticker symbol  
- Sector  
- Industry  
- Country  
- Currency  
- Market capitalization  

### Usage
- Screener filters  
- Sector‑based comparisons  
- Linking fundamentals and prices to companies  

---

## 4. Sector Benchmarks

### Source
External sector‑level financial datasets or aggregated internal calculations.

### Data Retrieved
- Average P/E ratio per sector  
- Average EV/EBITDA  
- Average price‑to‑book  
- Average FCF yield  

### Usage
- Relative valuation  
- Hybrid valuation weighting  
- Sector‑level comparisons  

---

## 5. Internal Data (Stored in PostgreSQL)

### Tables
- `companies`  
- `fundamentals`  
- `prices`  
- `valuations`  
- `sector_benchmarks`  
- `jobs_log`  

### Usage
- Persistent storage for all ETL results  
- Input for valuation engine  
- Data served to frontend via API  

---

## 6. Derived Data (Generated by the System)

### Examples
- DCF fair value  
- Relative fair value  
- Hybrid fair value  
- Discount percentage  
- Screener ranking score  

### Usage
- Screener output  
- Detail pages  
- Historical valuation tracking  

---

## Summary

The platform integrates three categories of data:

1. **External market and fundamentals data**  
2. **Internal processed and normalized data**  
3. **Derived valuation outputs**

Together, these sources enable accurate valuation, screening, and visualization of undervalued stocks.
